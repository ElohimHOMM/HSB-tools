## Node Stuff:

## VERSIONING:
https://semver.org/

tl;dr
Major.Minor.Patch

e.g. 0.1.0 -> Major Patch 0, Minor Patch 1, Patch 0
- Ideally after every new feature + 1 on minor patch
- after big changes a major version
- after a bugfix etc. +1 on patch
